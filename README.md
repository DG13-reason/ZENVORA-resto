WELCOME ZENVORA-resto PROJECT
Zenvora-resto merupakan sebuah desain website restoran yang digunakan oleh konsumen untuk membantu pemesanan di tempat maupun di rumah atau berada jauh dari restoran
pada website ini terdapat beberapa fitur
1. login
2. register
3. keranjang
4. pesan
5. Rating dan Ulasan
6. metode pembayaran (e-wallet, QRIS, Transfer bank, Tunai, COD(pesanan antar))
Fitur Admin
1. login
2. melakukan CRUD

ZENVORA-resto mewajibkan untuk login terlebih dahulu karena jika memilih menu akan muncul pop-up login jika belum login

Alur Login dan register
-login-
1. menekan logo profile pada navbar
2. muncul pop-up login
3. input email/username dan password
4. register jika belum
5. jika sudah tekan button login

-Register-
1. menekan register pada pop-up login
2. pop-up register muncul
3. input username
4. input email
5. input password
6. upload photo profile (opsional)
7. button Register

jika sudah login pengguna dapat melakukan pencarian menu dengan menginput menu yang diinginkan pada input searching
1. klik input searching menu
2. ketik menu yang diinginkan
3. akan pindah ke halaman menu muncul apa yang dicari

dibawah input search terdapat banner pengumuman seperti diskon, lalu dibawahnya terdapat menu rekomendasi
menu rekomedasi muncul apabila syarat ulasan dan rating pada menu tercapai
-rekomendasi menu terbagi menjadi 3
 1. main course
 2. drinks
 3. snack & dessert

begitu pun halaman menu terdapat 3 kategori seperti rekomendasi menu
alur pemesanan
-Dari mengklik pesan-
1. klik pesan pada menu yang diinginkan
2. muncul halaman payments
3. pilih metode reservasi (makan di restoran / kirim ke rumah)
   3.1 makan di restoran
       >input nama lengkap
       >input nomor Telepon
       >tanggal reservasi
       >jumlah orang
       >pilih area (indoor | outdoor | VIP)
       >Pilih nomor meja (berbeda setiap kategori area)
       >input catatan
       >Pilih metode pembayaran
             - tunai (tidak perlu upload bukti pembayaran)
             - Transfer (perlu upload bukti pembayaran)
             - E-Wallet (perlu upload bukti pembayaran)
             - QRIS (perlu upload bukti pembayaran)
   3.2 Kirim ke rumah
       >input nama lengkap
       >input nomor Telepon
       >input alamat
       >Kirim link google maps alamat
       >Pengantaran kurir resto
       >input catatan
       >Pilih metode pembayaran
             - COD (tidak perlu upload bukti pembayaran)
             - Transfer (perlu upload bukti pembayaran)
             - E-Wallet (perlu upload bukti pembayaran)
             - QRIS (perlu upload bukti pembayaran)
4. Mengisi ulasan dan rating pada history di halaman profile

Pada halaman profile user dapat mengedit dan melakukan logout

UNTUK ADMIN memilik dashboard sendiri
1. klik pada navbar menu bagian dasbord
2. Muncul menu
   - Kelola Menu (bisa CRUD)
   - Kelola Pesanan (update status dan melihat detail pesanan)
   - Kelola User
   - Kelola Meja (Melihat status dan CRUD)

itu saja yang ada pada website ini kedepannya mungkin ada pembaruan.
